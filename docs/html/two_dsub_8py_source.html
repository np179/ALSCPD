249  '''Function to find the number of relevant singular values and vectors for a given
250  SVD by comparing the sum of the squared neglected singular values against a threshhold.
251 
252  [Args]:
253  S[array]: Array of shape (r,s) containing the singular values along the second axis.
254  thresh[float]: Threshold to signal convergence, default is 1E-4. This value was chosen
255  by running multiple tests and adjusting until no inconsistant behaviour was
256  observed.
257 
258  [Returns]:
259  [int]: Number of relevant singular values and vectors wrt threshold.'''
260 
261  SVrel = 1
262  error = max([(S[i,SVrel:]**2).sum() for i in range(S.shape[0])])
263  while error > thresh:
264  if SVrel + 1 <= S.shape[1]:
265  SVrel += 1
266  error = max([(S[i,SVrel:]**2).sum() for i in range(S.shape[0])])
